Theme Features

  1. Full WordPress 4.1.x Compatibility
  2. Easy installation
  3. bbPress
  4. Visual Composer
  5. WooCommerce
  6. WPML
  7. Revolution Slider
  8. Right To Left
  9. 5 Theme Colors
  10. Infinite button
  11. Display Video(Youtube ID, ViMeo ID, video with HTML5), Slide, Image
  12. Responsive design
  13. Bootstrap integrate
  14. Custom meta boxes
    • Post/Page Meta box
  15. Built-in shortcodes
  16. Many background patterns available
  17. Translation with .mo/.po files
  18. Advanced theme options panel (Theme Options Section)
  19. Mega Menu
  20. Different header styles
